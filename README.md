# micor1-todo-list

a simple to-do list app.

> Create tasks.

> Delete tasks.

> Tag tasks.

## Built with Next, Tailwind and Shadcn/ui

## Install

```sh
cd .\todo\
npm install
```

## Usage

```sh
cd .\todo\
npm install
npm run dev
```

Spins up local servers on port 3000


Example:

```sh
Local:     http://localhost:3000
```
